RecipeIT Endpoints

    GET

  • /recipes - get all recipes
  • /ingredients - get all ingredients
  • /units - get all units
  • /properties - get all properties
  • /packs - get all packs

    POST

  • /recipes - post a new recipe. Receives a JSON file like {newRecipe: name: 'name', tags: 'tag1, tag2...', ingredients: 'ing1, ing2...' steps: 'step1, step2...', properties: 'prop1, prop2...'}
  • /properties - post new properties. Recieves a JSON file like {properties: 'prop1, prop2...'}
